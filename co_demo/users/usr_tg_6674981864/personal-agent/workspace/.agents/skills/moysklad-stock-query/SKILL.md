# Остатки по складу (МойСклад)

Получить остаток конкретной позиции на конкретном складе через МойСклад API. Выполняется в два шага: сначала поиск товара по коду, затем запрос остатков с фильтром по складу и товару.

## When to use
Когда пользователь спрашивает остаток по коду товара и/или названию склада. Примеры: «сколько 01500 на Комсомола?», «остаток позиции 00520 на складе Домостроительная».

## Steps
1. 1. Найти товар по коду: GET entity/product?filter=code={КОД} — получить id и name товара.
2. 2. Найти склад по названию: GET entity/store?search={НАЗВАНИЕ_СКЛАДА}&limit=10 — получить id и полное имя склада.
3. 3. Запросить остаток: GET report/stock/all?groupBy=product (с фильтром product={href_товара}&store={href_склада}) — вернуть stock, reserve, inTransit, quantity.
4. 4. Если href товара недоступен (скрыт политикой), использовать обходной путь: GET report/stock/all?search={ТОЧНОЕ_ИМЯ_ТОВАРА}&groupBy=product&limit=5 — из результата взять строку с совпадающим code и прочитать поля stock/quantity.
5. 5. Сообщить пользователю: название товара, склад, остаток (stock), в резерве (reserve), в пути (inTransit), доступно (quantity = stock - reserve).

## Success evidence
- Получен объект с полями stock и quantity для нужного товара.
- Название склада в ответе совпадает с запрошенным.
